## 面向汉语母语日语学习者的中日汉字字形辨析辅助教学系统

### 1. 项目概述
**漢字探偵 (Kanji Detective)** 是一款面向大学日语课堂的交互式教学辅助工具。
本项目旨在帮助汉语母语学习者识别中日汉字之间细微而容易被忽视的字形差异，减少因“同源异体字”所造成的认知干扰。

通过高频次的视觉辨析练习，学习者可以逐步增强对中日汉字笔画、结构和部件差异的敏感度，
例如“亚/亜”“发/発”“强/強”等，从而降低在日语书写与阅读过程中受到汉语字形负迁移影响的可能性。

### 2. 教学背景与目标

在日语学习过程中，汉语母语者往往容易因为汉字知识的既有优势而产生“看得懂就会写”的错觉，进而忽略日语标准字形与现代汉语规范汉字之间的差异。
这种过度依赖母语汉字知识的倾向，可能导致学习者在书写、输入和校对时混用中日字形。

本系统基于对比语言学与第二语言习得相关理念，旨在通过可视化、互动式的练习方式，帮助学习者建立更清晰的日语汉字字形表征。
具体目标如下：

- **增强字形觉察能力**：帮助学习者准确识别中日汉字在断笔、连笔、笔画方向及偏旁部首上的细微差异。
- **提供即时纠错反馈**：通过并置展示“误用字形”与“日语标准字形”，帮助学习者明确差异点。
- **实现难度递进训练**：通过动态难度系统，根据学习者的表现逐步提高任务难度，适应从基础词汇到完整语境的教学需求。

### 3. 核心功能模块 

#### 3.1 矩阵辨析模式

系统会生成由相似汉字组成的随机矩阵，学习者需要在字形高度相近的汉字阵列中找出符合日语规范的字形。
该模式主要训练学习者对单个汉字字形的快速识别能力，适合作为课堂导入或短时间集中练习使用。

#### 3.2 语境校勘模式

系统将目标汉字放入完整的日语句子中，学习者需要在阅读语境的同时识别其中混入的“中国语式汉字”误用。
该模式模拟真实的日语写作与校对场景，到帮助学习者将字形辨析能力迁移实际语言使用中。

#### 3.3 诊断反馈系统

当学习者选择错误时，系统会显示对比反馈弹窗，以可视化方式突出误用字形与正确字形的差异。
例如，系统会用红色标示错误字形，用绿色标示日语标准字形，帮助学习者从单纯的“判断正误”进一步发展到“理解差异”。

### 4. 技术规格与特点

- **跨平台交互**：基于 HTML、现代CSS动画与 JavaScript 实现 ，可在 Chrome、Edge、Safari 等主流浏览器中运行，适用于多种课堂设备环境。
- **动态难度调整**：系统会根据学习者的正确率与连续答对次数调整任务难度与限时设置，使练习更符合学习者当前水平。
- **自定义设置**：支持练习时长、音效开关等设置，便于教师根据课堂时间与教学目标灵活调整。

### 5. 部署与使用
- **运行环境**：支持 Chrome、Edge、Safari 等主流 Web 浏览器。
- **启动方式**：直接打开 `index.html` 即可进入系统主界面。
- **课堂使用建议**：可作为课堂导入、单元复习或竞赛型练习使用，也可用于课后自主训练。

## 中国語を母語とする日本語学習者のための中日漢字字形弁別支援システム

### 1. プロジェクト概要

**漢字探偵 (Kanji Detective)** は、大学の日本語教育現場での利用を想定したインタラクティブな学習支援ツールです。
本プロジェクトは、中国語を母語とする日本語学習者が、日中漢字の微細な字形差に気づきにくいという問題に着目し、類形語による認知的干渉を軽減することを目的としています。

高頻度の視覚的な弁別練習を通じて、学習者は「亚/亜」「发/発」「强/強」などに見られる筆画・構造・部首の違いに対する感度を高めることができます。これにより、日本語の読解や書字における中国語字形からの負の転移を抑えることを目指します。

### 2. 教育的背景と目的

中国語母語話者は、漢字知識をすでに有しているため、日本語の漢字学習において一定の優位性を持っています。
一方で、その既有知識に依存しすぎることで、日本語の標準字体と現代中国語の規範漢字との違いを見落としやすいという問題もあります。
その結果、書字、入力、校正の場面で日中字形の混用が生じる可能性があります。

本システムは、第二言語習得研究の知見を踏まえ、視覚的かつインタラクティブな練習を通じて、学習者が日本語漢字の字形をより明確に認識できるよう支援します。
具体的な目的は以下の通りです。

- **字形認識の強化**：筆画の接続・分離、筆順に由来する形の違い、部首の差異などを正確に識別できるようにする。
- **即時フィードバックの提供**：誤用字形と日本語の標準字形を並置して示すことで、学習者が差異を明確に把握できるようにする。
- **段階的な難易度調整**：学習者の正答率や連続正答数に応じて難易度を調整し、基礎的な語彙レベルから文脈内での弁別へと段階的に練習を進められるようにする。

### 3. 主要機能

#### 3.1 グリッド弁別モード

システムは、類似した漢字からなるランダムなグリッドを生成します。
学習者は、その中から日本語の標準字形に合致する漢字を探し出します。
このモードは、単一漢字の字形を瞬時に識別する力を養うことを目的としており、授業の導入や短時間の集中練習に適しています。

#### 3.2 文脈校正モード

ターゲットとなる漢字を日本語の文の中に配置し、学習者が文脈を読み取りながら、混入している「中国語式漢字」の誤用を見つけるモードです。
実際の日本語作文やレポート作成時の校正場面を想定しており、字形弁別能力を実際の言語使用へと結びつけることを目指します。

#### 3.3 診断フィードバックシステム

誤答時には、比較フィードバック用のポップアップが表示されます。
誤用字形を赤色、日本語の標準字形を緑色で示すなど、視覚的に差異を強調することで、学習者が単に正誤を判断するだけでなく、どこがどのように異なるのかを理解できるようにします。

### 4. 技術仕様と特徴

- **マルチデバイス対応**：HTML5 と JavaScript モダン CSS アニメーションを用いて実装されており、Chrome、Edge、Safari などの主要ブラウザで利用できます。教室内のさまざまな端末環境に対応します。
- **動的難易度調整**：正答率や連続正答数に基づき、問題の難易度や制限時間を自動的に調整します。
これにより、学習者の現在の習熟度に応じた練習が可能になります。
- **カスタマイズ設定**：授業時間や学習目的に応じて、練習時間や音響効果のオン・オフなどを調整できます。

### 5. 導入と使用方法

- **推奨環境**：Chrome、Edge、Safari などの主要 Web ブラウザ。
- **起動方法**：`index.html` を開くと、システムのメイン画面が起動します。
- **授業での活用例**：授業冒頭のウォームアップ、単元終了時の復習、競争形式の練習活動、または課後の自主学習として活用できます。
