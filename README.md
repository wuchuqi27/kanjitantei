## 面向汉语母语者的中日汉字字形辨析辅助教学系统

### 1. 产品概述 (Project Overview)

**漢字探偵 (Kanji Detective)** 是一款专为大学日语课堂设计的交互式教学辅助工具。本项目旨在解决汉语母语学习者在日语学习中因"同源异体字"（Cognate Variants）产生的认知干扰。通过高频次的视觉辨析练习，强化学习者对中日汉字细微笔画差异（如：亚/亜、发/発、强/強）的敏感度，从而降低书写及阅读中的负迁移影响。

### 2. 教学背景与目标 (Pedagogical Background & Objectives)

在对比语言学与二语习得领域，汉语母语者常因"过度泛化"倾向，忽视中日汉字在标准化字体（新字体 vs 规范汉字）上的结构差异。本产品基于认知对比分析理论，通过以下目标提升教学效果：

- **形体觉察增强**：精准识别中日汉字在断笔、连笔、偏旁部首上的细微差别。
- **即时纠错机制**：通过"误用展示"与"正确形体"的并置对比，构建清晰的心理表征。
- **难度梯度演进**：通过动态难度系统（Leveling System），适应从基础词汇到复杂句式辨析的教学进度。

### 3. 核心功能模块 (Core Functional Modules)

#### 3.1 矩阵辨析模式 (Grid-based Discrimination Mode)

系统生成基于随机矩阵的视觉搜索任务，要求用户在高度相似的汉字阵列中检索出符合日语规范的字形。此模式侧重于考察学习者对单一字形的瞬间识别能力。

#### 3.2 语境校勘模式 (Sentence-based Correction Mode)

将目标汉字置于完整的日语语义环境中。学习者需在句式阅读过程中，识别出由于输入法习惯或思维定势导致的"中式汉字"误用，模拟真实的学术书写纠错场景。

#### 3.3 纠错反馈系统 (Diagnostic Feedback System)

当发生辨析错误时，系统将触发对比教学弹窗，以视觉化的方式（红色标注错误字形，绿色标注标准日语字形）展示差异点，实现从"结果判断"到"知识内化"的转化。

### 4. 技术规格与特性 (Technical Specifications)

- **跨平台交互**：基于 HTML5 与现代 CSS 动画引擎，适配多终端课堂环境，确保流畅的交互体验。
- **动态难度算法**：系统根据用户的得分率（Accuracy）与连击数（Combo）实时调整挑战难度与限时，符合克拉申"i+1"输入假说。
- **自定义配置**：支持教学时长调整及音效开关，以适应不同强度的课堂教学需求。

### 5. 部署与使用 (Deployment & Usage)

- **环境要求**：支持主流 Web 浏览器（Chrome, Edge, Safari）。
- **启动方式**：直接访问 `index.html` 即可进入系统主界面。
- **课堂建议**：建议作为课堂导入（Warm-up）或单元复习阶段的竞争性游戏使用。

## 中国語を母語とする日本語学習者のための漢字字形弁別支援システム

### 1. 製品概要

**漢字探偵 (Kanji Detective)** は、大学の日本語教育現場に特化したインタラクティブな学習支援ツールです。本プロジェクトは、中国語母語話者が日本語学習において直面する「同源異体字」による認知的干渉の解決を目的としています。高頻度の視覚的弁別トレーニングを通じて、中日漢字の微細な字体差（例：亚/亜、发/発、强/強）に対する感度を高め、書字および読解における負の転移を軽減します。

### 2. 教育的背景と目標

対照言語学および第二言語習得理論において、中国語母語話者は「過剰般化」の傾向により、日中両国における標準字体（新字体 vs 規範漢字）の構造的差異を軽視しがちです。本製品は対照分析仮説に基づき、以下の目標を掲げています：

- **字形認識の強化**：筆画の接続や偏旁部首における微細な差異を正確に識別する。
- **即時フィードバック**：誤用と正解字形を並置することで、正しいメンタル・レプリゼンテーションを構築する。
- **難易度の段階的進化**：動的難易度調整システムにより、初級語彙から複雑な文脈での弁別まで、教授法に合わせた進展を可能にする。

### 3. 主要機能モジュール

#### 3.1 グリッド弁別モード (Grid-based Discrimination Mode)

ランダムに生成された行列の中から、日本語の規範に合致する字形を検索するタスクです。単一字形の瞬間的な識別能力を養うことに焦点を当てています。

#### 3.2 文脈校正モード (Sentence-based Correction Mode)

ターゲットとなる漢字を日本語の文章内に配置します。学習者は文を読み進める中で、思考の癖や入力ミスによる「中国語式漢字」を特定し、実際の学術的なライティングにおける推敲プロセスをシミュレートします。

#### 3.3 診断フィードバックシステム (Diagnostic Feedback System)

誤答時には、比較提示モーダルが起動します。視覚的な強調（誤字を赤、正解を緑で表示）により、差異を明確化し、「判定」から「理解」への深化を促します。

### 4. 技術仕様と特性

- **マルチデバイス対応**：HTML5とモダンCSSアニメーションを採用し、教室内の多様なデバイスでスムーズな操作を実現しています。
- **動的難易度アルゴリズム**：正解率（Accuracy）とコンボ数（Combo）に基づき、制限時間や難易度をリアルタイムで調整します（クラッシェンの「i+1」理論に準拠）。
- **カスタマイズ設定**：授業時間やニーズに合わせて、制限時間や音響効果の調整が可能です。

### 5. 導入と使用方法

- **推奨環境**：主要なWebブラウザ（Chrome, Edge, Safari）。
- **起動方法**：`index.html`を実行することで、システムが起動します。
- **活用提案**：授業の導入（ウォームアップ）や復習段階でのゲーミフィケーション要素として活用することを推奨します。
